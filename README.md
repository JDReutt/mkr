## Installation

Just add it to your ``rebar.config`` deps:

```erlang
  {deps, [
    ...
    {mkr, ".*",
      {git, "git@github.com:pzel/mkr.git", "master"}}
      ]}.
```

And then do the usual:

```bash
    $ rebar get-deps
    $ rebar compile
```
